# Instagram自动发布助手

## 项目介绍
本项目是一个自动化工具，用于按计划自动在Instagram上发布图片和文案。该工具能够按照预定的时间安排，自动执行文件管理、定时发布和Instagram操作，大大简化了社交媒体内容管理工作流程。

## 功能特性

### 1. 文件管理
- 按照规定结构存放于media/文件夹下
- 每周一自动删除media下的所有文件夹，然后创建新的文件夹结构
- 创建格式为`media/MMDD-MMDD`的主文件夹（表示一周的日期范围）
- 在主文件夹内创建Monday、Tuesday、Wednesday、Thursday、Friday子文件夹
- 用户手动在对应日期的文件夹中存放图片文件(.jpg)和文案文件(.txt)

### 2. 定时发布
- 周一至周五自动在12:00-14:00之间随机选择发布时间
- 每天自动触发一次发布流程

### 3. Instagram自动发布
- 自动打开Instagram网站
- 自动登录用户账号
- 自动上传对应日期文件夹中的图片和文案
- 自动点击发布按钮完成操作

## 技术栈
- Python 3.x
- Selenium（用于浏览器自动化）
- APScheduler（用于定时任务调度）
- 标准库：os, datetime, random（用于文件操作和时间管理）

## 安装说明
1. 克隆或下载本项目到本地
2. 安装所需依赖：`pip install selenium apscheduler`
3. 下载对应浏览器的WebDriver（如ChromeDriver）并配置到系统PATH
4. 在项目根目录创建配置文件，填写Instagram账号信息

## 使用说明
1. 在每周一之前，将需要发布的图片和文案分别放入对应的日期文件夹中
2. 运行主程序：`python main.py`
3. 程序将在后台运行，自动按计划执行发布任务

## 注意事项
- 请确保电脑在预定发布时间内保持开机状态
- 建议不要在发布期间关闭程序或锁定电脑
- 如遇网络问题或Instagram网站更新，可能需要调整自动化脚本
- 使用该工具时请遵守Instagram的使用条款，避免账号被限制或封禁
